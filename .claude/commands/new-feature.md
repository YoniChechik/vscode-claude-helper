# New Feature Setup

Creates a new feature branch using git worktree for isolated development with full planning.

## Feature description from user input
"$ARGUMENTS"

### Feature Description Validation
  - If empty or missing: "Error: Feature description is required. Please provide a detailed description for planning."

## Process

### Step 1: Create Worktree
Run /create-worktree command to set up isolated feature branch

### Step 2: Plan
Run /plan-feature command to create the feature plan

**FROM NOW ALL NEW WORK SHOULD ONLY BE DONE IN THIS FEATURE DIR**
